# PONG

![Pong](screenshot.png)

## Usage

```text
./pong.py [ai]  or  python pong.py

Options:

    - ai: left player is controlled by the computer (default: off)
```

## Controls

- Right player: up/down keys
- Left player: Q/A keys (if AI is off)
- Start: Space
- End: Escape

## Requirements

- Python 2.x
- PyGame
